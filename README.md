# Shopping List Telegram Bot 

Telegram-бот на Ruby для управления списком покупок с использованием гема **Shopping List Manager**.

## Возможности

* Добавление товаров через Telegram
* Просмотр списка покупок
* Отметка товаров как купленных
* Удаление товаров
* Подсчёт общей стоимости
* Поддержка пользовательских файлов списков

---

## Установка

```bash
git clone https://github.com/TimofeyBN/TgBot_Shopping-list.git
cd tgBot_Shopping-list
bundle install
```

---

## Настройка

1. Создайте бота через BotFather в Telegram
2. Получите токен
3. Создайте файл `.env` или укажите переменную окружения:

```bash
TELEGRAM_BOT_TOKEN=your_token_here
```

---

## Запуск

```bash
bundle exec ruby bot.rb
```

---

## Команды бота

### Добавить товар

```
/add Яблоки 3 67.99
```

### Показать список

```
/list
```

### Отметить как купленный

```
/buy 1
```

### Удалить товар

```
/delete 1
```

### Итоговая стоимость

```
/total
```

---

## Как это работает

Бот использует гем **Shopping List Manager** для обработки логики:

* Хранение данных в JSON
* Управление товарами
* Подсчёты и операции со списком
